# GS-Stats
A tool to perform stats on genotype data based on sample and marker.
______________________________________________________________________________________________________________________________________________________________________________________________

ABOUT: 

Code is part of a module for a larger program. It can also work as a stand-alone tool. The program evaluates samples based on gender and age and performs statistical tests on mean call rates to determine if one set is significantly different than others.  More information to follow.  Run program with -h for instructions.
